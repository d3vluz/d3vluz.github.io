Bem-vindo ao repositório do meu currículo online! Este projeto é um currículo interativo, responsivo e construído com foco na clareza de informações e boa experiência do usuário.

## Sobre o Projeto

O site apresenta meu perfil profissional, experiências, formação acadêmica e minha stack tecnológica. A interface foi baseada no `Curriculum Vitae`, trazendo as seguintes funcionalidades:

- **Modo Claro/Escuro**: Comutador de tema flutuante no canto da tela, que respeita a configuração de tema do sistema operacional inicialmente e salva a escolha do usuário via `localStorage`.
- **Otimizado para Impressão**: Layout customizado para folhas A4. Ao tentar imprimir a página (ou salvar como PDF), o site remove automaticamente elementos desnecessários (como os botões flutuantes) e força o tema claro.

## Tecnologias e Ferramentas

Para manter o projeto simples, rápido e de fácil manutenção, foram utilizadas as seguintes tecnologias:

- **HTML5 & CSS3**
- **Vue.js 2 (CDN)**: Para o gerenciamento reativo dos dados do currículo e lógicas de estado da tela.
- **Bootstrap 4**: Sistema de grids e responsividade.
- **FontAwesome**: Biblioteca para ícones de contato e dos botões flutuantes.
